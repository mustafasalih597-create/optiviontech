Journal Restriction For Users

This module restricts journals for specific users. You can add access users on journal configuration, only allowed users can access that journal. Users are allocated in specific journals like invoice, bill, cash, bank, sale & purchase, So users can not access a journal where the journal is not available for that user.

Features

Restrict Specific Journals to Users

Limit which users can view or use specific accounting journals such as invoice, bill, cash, bank, sales, and purchase.

Assign Allowed Users per Journal

Configure allowed users on each journal so only authorized users can access and transact with it.

Assign Allowed Journals per User

Define permitted journals on user profiles so users can work only with assigned journals.

Security Group Based Control

Enable a security group to activate journal restriction options in user and journal settings.

Administrator Unrestricted Access

Administrators can view and use all journals regardless of restrictions for full control and oversight.

Warning on Unauthorized Journal Use

Show a warning or alert when a user attempts to use or select an unauthorized journal.

Simple Configuration Inside Standard Views

Configure journal restrictions directly from standard user and journal forms without extra setup.
